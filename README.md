# Interpretable Voice Scoring via Zero-Shot Semantic Expansion

This repository contains the codebase and resources for evaluating clinical voice pathology using Audio Spectrogram Transformers (AST) and Orthogonal Procrustes alignment. By bridging acoustic latent spaces with semantic language models, this tool provides continuous, interpretable voice scoring grounded in clinical vocabulary.

## Repository Structure

This project is divided into two main components. Please navigate to the respective directories for detailed documentation, setup instructions, and execution scripts:

* **[`/research`](./research):** Contains the core machine learning pipeline. This includes the model weights, Orthogonal Procrustes alignment matrices, and a minimal reproducible example demonstrating the zero-shot semantic expansion described in the manuscript.
* **[`/app`](./app):** Contains the source code for the web application built to deploy the voice analysis tool.
